# 1. Función

Utilidad para dividir rangos de IPs en bloques de tamaño definido por el usuario. Acepta un único rango o un bloque de rangos.

# 2. Como utilizar

* Utiliza **DivideIP.htm** si unicamente deseas dividir un rango de IPs en rangos de tamaño definido.

![Imagen mostrando aplicación 1](/screenshots/imagen1.png)

* Utiliza **DivideIP_2.htm** si deseas dividir diferentes rangos de IPs en grupos de rangos de tamaño definido.

![Imagen mostrando aplicación 2](/screenshots/imagen2.png)

No se precisan librerias adicionales.
